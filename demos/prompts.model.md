```bash
#

#

sum_numbers() {
    # If no arguments were given, the sum is zero.

    # Use awk for portable floating‚Äëpoint arithmetic.
    # It reads the arguments as a space‚Äëseparated list.
    awk '

}
```

2. **Arithmetic** ‚Äì Bash itself only does integer arithmetic with `$((‚Ä¶))`. To handle decimals (e.g., `5.2`) we delegate the math to `awk`, which supports floating‚Äëpoint numbers out of the box.
3. **`awk` script**  
   * `BEGIN { sum = 0 }` ‚Äì initialise the accumulator.  
   * The `for` loop walks through each argument (`$(i)`) and adds it to `sum`.  
   * `END { printf "%g\n", sum }` prints the final total, using `%g` to avoid trailing zeros (e.g., `7` instead of `7.0`).
