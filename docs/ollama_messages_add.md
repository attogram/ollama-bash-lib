# ollama_messages_add

Adds a message to the current chat context.

## Alias

`oma`

## Usage

```bash
ollama_messages_add "role" "message"
```

## Arguments

* `"role"`: The role of the message author (e.g., "user", "assistant").
* `"message"`: The content of the message.

## Output

None.

## Return

* `0`: Success.
* `1`: Error.
