# ollama_app_installed

Checks if the Ollama application is installed.

## Alias

`oai`

## Usage

```bash
if ollama_app_installed; then
  echo "Ollama is installed."
else
  echo "Ollama is not installed."
fi
```

## Output

None.

## Return

* `0`: Ollama is installed.
* `1`: Ollama is not installed.
