# ollama_show_json

Shows information about a model in JSON format.

## Alias

`osj`

## Usage

```bash
ollama_show_json "model_name"
```

## Arguments

* `"model_name"`: The name of the model to show.

## Output

A JSON object containing information about the model is sent to `stdout`.

## Return

* `0`: Success.
* `1`: Error.
