# ollama_api_get

Performs a GET request to the Ollama API.

## Alias

`oag`

## Usage

```bash
ollama_api_get "/api/path"
```

## Arguments

* `"/api/path"`: The API endpoint to query.

## Output

The result of the API call is sent to `stdout`.

## Return

* `0`: API call was successful.
* `1`: API call failed.
